# 跳跃表(SkipList)期望层高推导

## 1. 引言

跳跃表（SkipList）是一种概率性数据结构，由William Pugh于1989年提出。它在有序链表的基础上增加了多级索引，使得查找、插入和删除操作的平均时间复杂度为O(log n)。跳跃表的性能与其**节点层数分布**密切相关，而期望层高是分析跳跃表性能的关键参数。

## 2. 跳跃表基本结构

### 2.1 节点结构
每个跳跃表节点包含：
- 存储的值（Key）
- 一个**层高（Level）** 属性，表示该节点在多少层链表中出现
- 每层对应的**前进指针**数组，指向同层的下一个节点

### 2.2 概率参数p
跳跃表的核心机制是**随机层高生成**：
- 每个新节点以概率p提升到更高层
- 通常取p = 1/2或p = 1/e

## 3. 层高概率分布

### 3.1 定义
设随机变量L表示节点的层高（最底层为1层）。层高生成过程可看作一系列伯努利试验：

```
第1层：所有节点必然出现（概率=1）
第2层：以概率p出现
第3层：以概率p²出现
...
第k层：以概率p^(k-1)出现
```

### 3.2 概率质量函数
节点恰好有k层的概率：
```
P(L = k) = p^(k-1) * (1-p),  k = 1, 2, 3, ...
```

**推导**：
- 节点必须出现在1到k-1层（概率p^(k-2)）
- 出现在第k层（概率p^(k-1)）
- 不在第k+1层出现（概率1-p）
因此：P(L = k) = p^(k-1) * (1-p)

## 4. 期望层高计算

### 4.1 数学推导

期望层高E[L]定义为：
```
E[L] = Σ_{k=1}^∞ k * P(L = k)
     = Σ_{k=1}^∞ k * p^(k-1) * (1-p)
```

令q = p，展开求和：
```
E[L] = (1-p) * Σ_{k=1}^∞ k * p^(k-1)
```

利用幂级数求和公式：
```
Σ_{k=1}^∞ k * r^(k-1) = 1/(1-r)²,  |r| < 1
```

代入r = p：
```
E[L] = (1-p) * 1/(1-p)²
     = 1/(1-p)
```

### 4.2 常见p值对应的期望层高

| 概率p | 期望层高 E[L] | 说明 |
|------|--------------|------|
| 1/2  | 2.0          | 最常用值 |
| 1/3  | 1.5          |      |
| 1/e ≈ 0.3679 | ≈ 1.582 | 理论最优值 |
| 1/4  | 1.333        |      |

## 5. 推导验证

### 5.1 级数展开验证
当p = 1/2时：
```
E[L] = 1/(1-1/2) = 2
```

展开验证：
```
E[L] = 1*(1/2)⁰*(1/2) + 2*(1/2)¹*(1/2) + 3*(1/2)²*(1/2) + ...
     = 1/2 + 2/4 + 3/8 + 4/16 + 5/32 + ...
     = 0.5 + 0.5 + 0.375 + 0.25 + 0.15625 + ...
     ≈ 2
```

### 5.2 几何分布视角
节点的层高可视为几何分布：
- 每次"尝试提升一层"看作一次伯努利试验
- 失败（不再提升）的概率为(1-p)
- 试验次数（层高）的期望为1/(1-p)

## 6. 最大层高限制

### 6.1 实际实现中的约束
实际编程中通常设置最大层高MaxLevel：
```
MaxLevel = ⌈log_{1/p}(N)⌉ + 1
```
其中N是预期元素数量。

### 6.2 修正的概率分布
当存在最大层高M时：
```
P(L = k) = p^(k-1)(1-p),  k = 1, 2, ..., M-1
P(L = M) = p^(M-1)  (最后一层必然存在)
```

修正后的期望：
```
E[L] = Σ_{k=1}^{M-1} k * p^(k-1)(1-p) + M * p^(M-1)
```

## 7. 性能分析应用

### 7.1 空间复杂度
n个节点的期望总层数：
```
E[总层数] = n * E[L] = n/(1-p)
```
- 当p=1/2时，空间复杂度为O(2n) = O(n)

### 7.2 搜索时间复杂度
查找操作的期望比较次数：
```
E[比较次数] = (log_{1/p} n) / p + 1/(1-p) + 1
```
其中第一部分来自向下搜索，第二部分来自同层遍历。

## 8. 结论

1. **核心公式**：跳跃表节点的期望层高为 E[L] = 1/(1-p)
2. **常用取值**：p=1/2时，E[L]=2，空间开销约为2n
3. **理论最优**：p=1/e时，期望层高约1.582，空间效率最高
4. **实际权衡**：p值越小，层高越低，空间效率高但搜索慢；p值越大，搜索快但空间开销大

## 附录：数学推导细节

### A.1 无穷级数求和
```
S = Σ_{k=1}^∞ k * r^(k-1)
```
令T = Σ_{k=0}^∞ r^k = 1/(1-r)
对T求导：
dT/dr = Σ_{k=1}^∞ k * r^(k-1) = 1/(1-r)²
得证：S = 1/(1-r)²

### A.2 方差计算
层高的方差：
```
Var(L) = E[L²] - (E[L])²
E[L²] = Σ k² * p^(k-1)(1-p)
       = (1+p)/(1-p)²
Var(L) = p/(1-p)²
```
当p=1/2时，Var(L)=2，标准差≈1.414

---

**文档总结**：跳跃表的期望层高推导展示了概率数据结构的设计美学，通过简单的随机过程实现了平衡的性能表现。1/(1-p)的简洁形式使工程师能够直观地权衡时间与空间效率。
