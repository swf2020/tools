# DeepSeek 并发文档生成器

这是一个改进版的 DeepSeek API 调用脚本，支持并发处理多个主题并记录处理结果。

## 主要特性

1. **并发处理**：使用线程池并发调用 DeepSeek API，提高处理效率
2. **结果记录**：自动记录成功和失败的主题到本地文件
3. **进度显示**：实时显示处理进度和状态
4. **错误处理**：完善的异常处理机制

## 使用方法

1. 安装依赖：
```bash
pip install openai httpx
```

2. 配置 API Key：
   - 方法1：在代码中设置 `DEEPSEEK_API_KEY` 环境变量
   - 方法2：设置系统环境变量

3. 准备配置文件：
   - `topics.json`：包含要处理的主题列表
   - `SKILL.md`：提供文档生成的指导提示

4. 运行脚本：
```bash
python deepseek.py
```

## 配置说明

### topics.json 格式
```json
{
    "分类名称": [
        "主题1",
        "主题2",
        "主题3"
    ]
}
```

### 并发参数调整
在代码中修改 `MAX_WORKERS` 参数来调整并发数：
```python
MAX_WORKERS = 3  # 根据需要调整并发线程数
```

## 输出文件

脚本会生成以下文件：
- `主题名称-日期.md`：生成的技术文档
- `success_topics_时间戳.txt`：成功处理的主题列表
- `failed_topics_时间戳.txt`：处理失败的主题及错误信息

## 注意事项

1. 请确保网络连接正常
2. API 调用可能产生费用，请注意控制使用量
3. 建议先用少量主题测试后再批量处理
